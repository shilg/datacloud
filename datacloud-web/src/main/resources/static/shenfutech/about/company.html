公司介绍

上海申服信息科技有限公司成立于2015年,是一家以软件开发、系统集成、多媒体应用及融合通讯服务业务为主的高科技公司。 申服科技与上海无线通讯研究中心合作，研制开发了无线AP的云管理平台——蒲公英无线云管理系统DWCCS(Dandelion Wifi Cloud Control System) 和IntelliOps 智能一体化运维平台。这两项拥有自主知识产权的管理系统，作为申服科技的拳头产品推向市场。
