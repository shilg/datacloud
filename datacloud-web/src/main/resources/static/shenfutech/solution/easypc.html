HelpDesk好帮手
桌面故障自检、自愈、自上报工具
简介

IDC设备正常、应用正常，但用户会因PC环境、电脑配置等原因无法使用业务系统，针对有分公司的企业，找IT人员解决时效慢、IT人员定位问题难

核心功能

一键检测与修复、编排检测及修复步骤、可调用外部脚本，灵活扩展、自动收集资产信息、一键生成工单

运行方便

独立的EXE小程序，不需要安装，直接运行。

支持的系统

XP、WIN7、WIN8、WIN10、Win2003、Vista

详情查看

可查看每项任务的详情况，成功或失败。

自助引导

针对错误的任务，可以对用户进行提示，引导用户自助处理。

丰富组件

小程序集成了常用的检测与修复组件，方便用户调用。

任务自定义编排

IT管理员可根据公司不同角色需要的PC环境，编排不同的检测工具。

灵活扩展

除了官方自带的组件，可调用外部自己编写的bat脚本，灵活扩展。

灵活的提示信息

提示信息支持Html标签，可以跳转到帮助网站、文档下载页面。

自动运行

可设置小程序开机运行，定时上传本地数据，收集资产信息，获取一线客户端访问内部系统的性能数据。

自定义API

运行记录、登录信息、工单信息可使用我们的SaaS平台。可也填内部的ITSM的地址，快速对接。

终端数据收集

获取终端数据，可获取操作系统类型、IE版本、运营商线路等信息，做数据分析统计。

资产系统对接

与我们的资产系统整合对接、自动更新电脑的配置信息

事件查看

事件上报时，用户的检测数据一同提交，提升IT维护人员解决问题的速度。
